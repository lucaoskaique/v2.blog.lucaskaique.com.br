    # Bem-vindo ao Post Multilíngue

    Este é um exemplo de post que demonstra como o sistema de i18n funciona.

    ## Características

    - Suporte a múltiplos idiomas
    - Estrutura simples de arquivo único
    - Fácil de gerenciar no CMS

    ## Como funciona

    Quando você cria um post no Decap CMS, você verá abas para cada idioma configurado.
    Você pode alternar entre as abas e preencher o conteúdo em cada idioma.

    ### Campos traduzíveis

    Os seguintes campos podem ser traduzidos:
    - Título
    - Descrição
    - Corpo (conteúdo principal)

    ### Campos compartilhados

    Os seguintes campos são compartilhados entre todos os idiomas:
    - Data de publicação
    - Imagem
    - Tags
    - Categorias
    - Classe principal
    - Cor

    ## Conclusão

    Com esta estrutura, você pode facilmente criar conteúdo multilíngue para o seu blog!

    # Welcome to the Multilingual Post

    This is an example post that demonstrates how the i18n system works.

    ## Features

    - Multi-language support
    - Simple single-file structure
    - Easy to manage in CMS

    ## How it works

    When you create a post in Decap CMS, you'll see tabs for each configured language.
    You can switch between tabs and fill in content for each language.

    ### Translatable fields

    The following fields can be translated:
    - Title
    - Description
    - Body (main content)

    ### Shared fields

    The following fields are shared across all languages:
    - Publication date
    - Image
    - Tags
    - Categories
    - Main class
    - Color

    ## Conclusion

    With this structure, you can easily create multilingual content for your blog!
